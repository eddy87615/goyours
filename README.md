# Go Yours - 去你的打工度假

Go Yours（高優國際）是專業的日本留學代辦與打工度假諮詢服務平台，由一群熱愛日本文化的年輕人創立，致力於協助台灣人實現日本生活夢想。

## 我們的服務

### 日本留學代辦

- 日本語言學校申請與諮詢
- 學校選擇建議與比較
- 簽證申請協助
- 住宿安排與生活規劃

### 打工度假支援

- 打工度假簽證申請輔導
- 行前準備與注意事項
- 工作媒合與職缺資訊
- 在日生活經驗分享

### 日本就職服務

- 日本正職工作機會媒合
- 兼職打工資訊提供
- 履歷撰寫與面試指導
- 職涯發展諮詢

## 為什麼選擇 Go Yours？

- **專業團隊**：擁有豐富的日本留學與打工度假經驗
- **全程協助**：從申請到抵達日本，提供完整支援
- **真實分享**：提供第一手的在日生活經驗與資訊
- **用心服務**：以學員需求為優先，提供客製化建議

## 網站功能

- **最新消息**：日本留學與打工度假相關資訊
- **學校介紹**：詳細的日本語言學校資訊
- **工作機會**：日本正職與兼職職缺
- **線上諮詢**：便利的聯絡表單與即時諮詢

## 聯絡我們

- **官方網站**: [https://goyours.tw](https://goyours.tw)
- **服務時間**: 週一至週五 10:00-18:00
- **線上諮詢**: 透過網站聯絡表單或即時通訊

# Go Yours - 日本ワーキングホリデー・留学サポート

Go Yours（高優国際）は、日本留学代行およびワーキングホリデーコンサルティングサービスを提供する専門プラットフォームです。日本文化を愛する若者たちによって設立され、台湾の方々の日本生活の夢を実現するお手伝いをしています。

## サービス内容

### 日本留学代行

- 日本語学校の申請とコンサルティング
- 学校選択のアドバイスと比較
- ビザ申請サポート
- 宿泊施設の手配と生活プラン

### ワーキングホリデーサポート

- ワーキングホリデービザ申請サポート
- 渡航前準備と注意事項
- 仕事のマッチングと求人情報
- 日本生活体験シェア

### 日本就職サービス

- 日本正社員求人マッチング
- アルバイト情報提供
- 履歴書作成と面接指導
- キャリア開発コンサルティング
